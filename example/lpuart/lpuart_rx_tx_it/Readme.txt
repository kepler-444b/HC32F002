功能描述
================================================================================
本样例主要涉及串口中断收发数据样例

说明：
PA02: TX
PA01: RX
配置：9600、8bits、1stop

================================================================================
测试环境
================================================================================
测试用板:
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
串口线

辅助软件:
---------------------
串口调试工具

================================================================================
使用步骤
1、重新编译代码并下载
2、串口调试工具配置波特率9600bps、无校验、1stop，HEX格式显示
3、下载程序后，重新上电，PC串口收到0x0A、0x0D表示连接成功；
3、PC串口发送1字节数据[LEN](表示请求接收的数据长度)
4、发送完成后，收到[LEN]字节数据(从LEN ~ 1)

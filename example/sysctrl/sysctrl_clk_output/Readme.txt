================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器


辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并下载运行；
2） 连接PA01（HCLK输出）和PC04（PCLK输出）到示波器；
3） 重新上电后，按下USER KEY按键，程序开始运行，观察时钟输出；
4） 查看示波器显示频率；（HCLK = 4M；PCLK = 250K；）

注意
================================================================================
可改变样例时钟配置及输出类型，观测输出结果。

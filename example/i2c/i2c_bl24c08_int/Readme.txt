功能描述
================================================================================
本样例主要涉及I2C以中断的方式读写E2PROM功能

说明：
引脚设置：
PA02：SCL
PA01：SDA

================================================================================

SK_HC32F002_TSSOP24_V1.0
---------------------
（需要填）

---------------------

辅助软件:
---------------------
串口调试工具
================================================================================
使用步骤
1、PA02、PA01端口与E2PROM的SCL和SDA正确连接
2、重新编译，使用debug下载运行，并按下User按键
3、在调试窗口观察数组u8Recdata的值是否和数组u8Senddata的值一致，如果一致，运行正常

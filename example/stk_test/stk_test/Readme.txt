功能描述
================================================================================
本样例主要演示STK基本功能

说明：
PA02: TX发送
配置：9600、8bits、1stop

================================================================================
测试环境
================================================================================
测试用板:
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
串口线

辅助软件:
---------------------
串口调试工具

================================================================================
使用步骤
1、重新编译代码下载；
2、串口调试工具配置波特率9600bps、无校验、1stop,字符串显示模式；
3、确认串口连接及配置无误；
4、重新上电，窗口显示MCU相关信息；
5、默认状态或释放USERKEY，LED灯快速闪烁；
6、按下USERKEY，LED慢速闪烁；

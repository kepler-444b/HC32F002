功能描述
================================================================================
本样例主要展示MCU的低电压检测（LVD）的上升沿和下降沿中断功能

说明：
调整PA03输入电压，触发LVD的中断

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  连接PA00到直流电源，直流电源输出2.4v以上。
2）  打开工程编译并运行。
3）  调整直流电源到2.4V以下(因PA03为负端，故此时表现为上升沿触发)。
4）  LED点亮，表示触发上升沿中断。
5）  调整直流电源到2.4V以上(因PA03为负端，故此时表现为下降沿触发)。
6）  LED再次点亮，表示触发下降沿中断。
================================================================================
注意
================================================================================
测试过程中也可以用示波器同步观测LVD OUT（PA02端口的电平变化）

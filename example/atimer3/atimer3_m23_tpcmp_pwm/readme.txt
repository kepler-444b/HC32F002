功能描述
================================================================================
本样例展示Atimer3的CHA与CHB输出独立PWM的双点比较功能，
使用CCRxA和CCRxB控制OCREFA（CHxA）的比较输出功能（双点比较功能，当CCRxA和CCRxB的值不相等时候，CHxA波形为非对称PWM）,
CCRxB控制OCREFB（CHxB）的比较输出功能

================================================================================

================================================================================

---------------------
示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器探头连接到PD05(ATIM3 CH0A)和PB00(ATIM3 CH0B), PD03(ATIM3 CH1A)和PB01(TIM3 CH1B),
   PD02(ATIM3 CH2A)和PB02(ATIM3 CH2B)
4）运行，观察波形输出
5）设置重复周期计数RCR为0，使能UDF下溢重复计数，每个周期的下溢点产生Uev触发中断，
并且改变输出占空比（可通过LED引脚（PB04）观察该中断的产生时机）
CHxA与CHxB的占空比相隔一个周期更替变化。
可以看到，CHxA的上升沿时机由CCRxA设定的值控制，CHxA的下降沿时机由CCRxB设定的控制。

================================================================================
注意
================================================================================
1）该样例展示的是CHxA与CHxB输出独立PWM的双点比较功能（CHxA的左右边沿分别CCRxA和CCRxB设置）

2）如果要测试CHxA与CHxB的互补PWM功能（双点比较：CHxA占空比由CCRxA和CCRxB共同设置，CHxB与CHxA互补），
请将stcTim3BaseCfg.u32PWMTypeSel  = ATIM3_M23_M23CR_COMP_PWM_INDEPT; ==》改为 stcTim3BaseCfg.u32PWMTypeSel  = ATIM3_M23_M23CR_COMP_PWM_COMP;

3）如果要测试CHA与CHB的互补PWM功能（单点比较：CHA占空比由CCR0A设置，CHB与CHA互补）
请将stcTim3BaseCfg.u32PWMTypeSel  = ATIM3_M23_M23CR_COMP_PWM_INDEPT; ==》改为 stcTim3BaseCfg.u32PWMTypeSel  = ATIM3_M23_M23CR_COMP_PWM_COMP;
stcTim3BaseCfg.u32PWM2sSel    = ATIM3_M23_M23CR_PWM2S_COMPARE_DOUBLE_POINT; ==》改为  stcTim3BaseCfg.u32PWM2sSel    = ATIM3_M23_M23CR_PWM2S_COMPARE_SINGLE_POINT;

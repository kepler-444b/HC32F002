功能描述
================================================================================
本样例主要展示Atimer3的外部输入计数功能。
说明：
本样例使用定时器计数外部输入信号，定时溢出后产生中断，并重新载入初值进行定时，并再次溢出进入中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器或者信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）外部脉冲1KHz信号通过ATIM3_ETR(PC05)输入。
4）重载模式下，ATIM3从ARR计数到溢出，会产生溢出中断，Atimer3计数1000次产生1S左右的时间间隔中断。
5）在中断内对LED(PB04)输出进行翻转，控制LED灯亮灭；

功能描述
================================================================================
本样例展示Atimer3的BK端口输入作为PWM（带硬件死区互补PWM输出）刹车功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
双通道示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器探头连接到PD05(ATIM3 CH0A)和PB00(ATIM3 CH0B), PD03(ATIM3 CH1A)和PB01(TIM3 CH1B),
   PD02(ATIM3 CH2A)和PB02(ATIM3 CH2B)
4）PB05接高电平
5）运行，观察波形输出（六路两两对称的互补PWM，并且带有死区时间）;
6）设置重复周期计数RCR为0，使能UDF下溢重复计数，每个周期的下溢点产生Uev触发中断，
并且改变输出占空比（可通过LED引脚（PB04）观察该中断的产生时机）
7) PB05接低电平，触发输入刹车，PWM输出为低。
8）若PB05重新接回高电平，4s后PWM恢复输出。

功能描述
================================================================================
本样例展示Atimer3的三角波作为载波的独立PWM输出功能

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器探头连接到PD05(ATIM3 CH0A)和PB00(ATIM3 CH0B), PD03(ATIM3 CH1A)和PB01(TIM3 CH1B),
   PD02(ATIM3 CH2A)和PB02(ATIM3 CH2B)
4）运行，观察波形输出（六路PWM）为中心对称;
5）设置重复周期计数RCR为0，使能UDF下溢重复计数，每个周期的下溢点产生Uev触发中断，
   并且改变输出占空比（可通过LED引脚（PA11）观察该中断的产生时机）

================================================================================
注意
================================================================================
1）如果要测试CHA与CHB的互补PWM功能（单点比较：CHA占空比由CCR0A设置，CHB与CHA互补）
请将stcAtim3BaseCfg.u32PWMTypeSel  = ATIM3_M23_M23CR_COMP_PWM_INDEPT;
==》改为  stcAtim3BaseCfg.u32PWMTypeSel  = ATIM3_M23_M23CR_COMP_PWM_COMP;

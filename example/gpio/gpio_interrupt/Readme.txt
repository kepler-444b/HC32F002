功能描述
================================================================================
本样例主要展示GPIO外部中断触发功能。
说明：
本样例使用USER KEY作为GPIO外部中断通过下降沿触发中断，进入中断服务程序，将
LED点亮延时200ms闪烁。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
万用表

辅助软件:
---------------------

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）将程序下载至Demo板；
3）重新上电;
4）按下USER KEY，触发外部中断，LED点亮并维持200ms；
5）LED点亮200ms后熄灭，程序正常运行。

================================================================================
注意
================================================================================
该Demo主要演示中断下降沿触发功能，其他方式的触发只需修改中断触发方式配置即可。

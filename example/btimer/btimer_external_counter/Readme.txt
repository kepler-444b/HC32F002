功能描述
================================================================================
本样例主要展示Btimer的计数器功能。
说明：
本样例对展示Btimer使用外部ETR输入脉冲作为计数时钟的计数器功能。Btimer3计数溢出产生中断，
并在中断内翻转LED。

================================================================================

---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器
信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）将PD06接信号发生器的输出
4）运行；
5）LED闪烁

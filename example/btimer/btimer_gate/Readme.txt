功能描述
================================================================================
本样例主要展示Btimer的门控功能。
说明：
本样例展示Btimer在门控模式下，定时器运行之后，ETR作为门控引脚：
当ETR输入为低的时候，Btimer运行，溢出后产生中断，并且在中断内翻转LED引脚电平，TOG输出; 
当ETR输入为高的时候，Btimer停止计数，LED不闪烁，TOG停止翻转。

无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器连接PA01和PA02
4）ERT(PD06)输入高电平，TOG翻转输出停止，LED不闪烁
   ERT(PD06)输入低电平，TOG翻转输出，LED闪烁

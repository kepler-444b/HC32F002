功能描述
================================================================================
本样例主要展示Btimer的级联功能。
说明：
本样例展示了Btimer3和Btimer4之间的级联功能。
Btimer3使用工作于定时器模式，
Btimer4工作于计数器模式，
Btimer3的时钟来自PCLK，计数到周期0xFFFF之后，溢出产生OV事件，输出给Btimer4，
作为Btimer4的计数时钟，Btimer4计数溢出后，产生溢出中断，并在中断内翻转LED电平。

功能描述
================================================================================
本样例主要展示MCU的看门狗的窗口功能。

说明：
在看门狗工作期间，如果喂狗发生在窗口期，则不会发生复位。
如果发身在非窗口期，或看门狗溢出则产生复位。

================================================================================

---------------------


辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）运行；
3）如果在窗口时间内按下按键喂狗(2s~5s)，则不会发生看门狗复位(LED不发生闪烁);
4）如果在窗口时间外按下按键喂狗(2s以内)，或溢出前没有喂狗，则复位(LED发生闪烁);

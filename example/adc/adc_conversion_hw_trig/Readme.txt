功能描述
================================================================================
本样例主要展示AD连续转换硬件触发功能


================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行；
2）PA01接外部模拟电压(要求低于AVCC)；
2）USER按键(PB03)作为外部触发信号，按下按键产生一个下降沿中断标志；
3）在主循环里面等待转换完成，读取(u32AdcRestult)转换结果,判断是否与PA01输入电压一致；



================================================================================
注意
================================================================================
转换结果=（输入电压/AVCC）*1024

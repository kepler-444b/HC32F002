功能描述
================================================================================
本样例主要展示ADC 单次采样软件启动转换示例，使用中断方式获取ADC运行状态和ADC的采样值。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）在ADC输入PIN(PA01)输入不同的电平。
2）打开工程编译并debug运行。
3）在ADC中断内读取转换值(u32AdcRestult),判断是否与PA01输入电压一致；。

================================================================================
注意
================================================================================
转换结果=（输入电压/AVCC）*1024

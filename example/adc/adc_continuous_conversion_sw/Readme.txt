功能描述
================================================================================
本样例主要展示ADC连续转换模式的应用案例。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）端口PA01接入直流电压，作为ADC的模拟输入信号;
2）打开工程编译并debug运行；
3）在中断里面等待转换完成，读取(u32AdcRestult)转换结果,判断是否与PA01输入电压一致；



================================================================================
注意
================================================================================
转换结果=（输入电压/AVCC）*1024
================================================================================

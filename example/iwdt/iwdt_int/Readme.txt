功能描述
================================================================================
本样例主要展示MCU的看门狗的中断及喂狗功能。

说明：
在看门狗工作期间，如果在看门狗溢出前未进行喂狗动作，可以产生中
断或复位。
================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK_HC32F002_TSSOP24_V1.0

辅助工具:
---------------------

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）运行；
3）如果在看门狗溢出前(约3.2s内)按下按键(喂狗)则LED不会点亮(溢出),否则LED被点亮(进入溢出中断);
